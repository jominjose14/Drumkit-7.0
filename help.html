Drum Kit 7.0



Best viewed on a 15" display


This drumkit can be played in two ways:

  • Mouse/Touch
  • Keyboard


Mouse-Clicks/Touch

In the first case, click or touch on the letter-buttons located next to each drum part to play it. Its as simple as that! Make sure you enable multi-touch in case of a touch-screen.


Keyboard

In the second case, before using the keyboard to play, check out the [Controls] page to understand the default key-bindings. The chosen keys are shown on the letter-buttons next to each drum-part. Press the linked up key on the keyboard to play the corresponding drum-part.

Enjoy!


Coming Feature

The feature of changing the default control keys to any key that the user likes, will soon be available. The [Update Controls] button is part of this feature. As of now, this button is not functional. DrumKit 7.0 remains under construction ;)


About

This is a free online drumkit web app, that is wired up using Javascript. Audio samples have been collected from sites like findsounds.com and samplefocus.com, and edited to suit the drumkit's feel and appeal.
